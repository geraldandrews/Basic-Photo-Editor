#Basic Image Editor
### Live Site: https://geraldandrews.github.io/Basic-Photo-Editor/

###Project Description: Personal
###Programs Used: HTML, CSS, and JavaScript
### Date Created: March 2026

## Description
Basic photo editor built from scratch that allows the user to upload a photo and change the brightness, saturation, inversion, and grayscale effects of the photo. Also lets the user rotate,
flip, and save the photo after all desired changes are complete.
